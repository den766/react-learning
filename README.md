# React Learning Lab

This repository contains small experiments built while learning React fundamentals.

The goal is to understand how React works internally before building larger applications.

Each experiment focuses on one concept such as state, events, component architecture, or side effects.

---

## Learning Focus

Core topics explored in this repository:

* React mental model
* Components and JSX
* Props and state
* Conditional rendering
* List rendering
* Event handling
* Hooks (`useState`, `useEffect`)
* Component architecture

---

## Repository Structure

src/
└ experiments
├ day1-counter
├ day2-toggle
├ day3-components
└ ...

Each folder contains a small experiment exploring a specific React concept.

---

## Purpose of This Repository

This repository is not a finished project.

It acts as a **React experimentation lab** used to:

* test concepts
* observe React rendering behavior
* build small UI patterns
* understand React architecture

Larger applications will be built in separate repositories.
